✓ TROUBLESHOOTING
===================

Q: "Build failed" in Visual Studio
A: Install Windows SDK via Visual Studio Installer
   - Open Visual Studio Installer
   - Select "Modify"
   - Check Windows 11 SDK
   - Click Modify

Q: "DLL not found" during installation
A: Ensure build succeeded. Check:
   Test-Path "c:\Source\ben-che\DiskBench\DiskBench.ShellExtension.Cpp\bin\Debug\DiskBench.ShellExtension.Cpp.dll"

Q: Context menu still doesn't appear
A: Clear cache and try again:
   .\Install-ExplorerCommand-Cpp.ps1 -Uninstall
   (wait 10 seconds)
   .\Install-ExplorerCommand-Cpp.ps1

Q: "Access Denied" error
A: Run PowerShell as Administrator
   (Right-click → "Run as Administrator")
